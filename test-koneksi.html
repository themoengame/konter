📋 Panduan Troubleshooting

  • Pastikan Web App sudah di-deploy (Deploy → New deployment → Web app)
  • Pastikan "Execute as" = Me (atau user yang punya akses ke sheet)
  • Pastikan "Who has access" = Anyone (untuk testing) atau Anyone with link
  • Cek di Apps Script Editor → Run → Test as web app
  • Jika kena CORS error, buka Web App URL langsung di browser terlebih dahulu untuk authorize
